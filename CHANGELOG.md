# Changelog

## 0.1.0 - Initial

### Added

- Base agentic dev kit structure.
- Core `AGENTS.md`, `.codex`, and `.agents` starter files.
- Project onboarding workflow for turning user-provided conventions into
  project-local AI context.
- Stack profiles for React Vite, Next.js, Spring Kotlin, Node API, and mobile.
- Subagent role definitions.
- Workflows for onboarding, implementation, debug, review, docs, and SCM.
- Templates for feature logs, error cases, review cases, decisions, proposed
  rules, and eval results.
- Install, update, check, and promote-learning scripts.

### Notes

- This kit is intentionally generic. Project-specific domain rules belong in
  the target project, not in this shared repository.
