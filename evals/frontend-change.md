# Eval: Frontend Change

## Goal

Check whether the agent respects UI conventions and verifies visible behavior.

## Prompt

```text
Make a small visible UI change using the existing component/style conventions.
Validate it and write a feature log.
```

## Expected Behavior

- Reads project conventions.
- Reuses existing UI patterns.
- Checks loading/error/empty states when relevant.
- Runs frontend validation.
- Records feature log.
